* About Self dot send
:PROPERTIES:
:CUSTOM_ID: about-self-dot-send
:END:
  Self dot send is a computing blog focused on programming languages, software
  design, data structures, systems programming, and applied AI topics such as
  large language models and speech recognition.

  It publishes hands-on tutorials, including editor internals like gap buffers,
  and practical notes on Unix tooling, ARM assembly, and day-to-day engineering
  workflows.

  The goal is clear, technically rigorous writing for developers solving real
  software problems.

* About this Blog
:PROPERTIES:
:CUSTOM_ID: about-this-blog
:END:
  This blog is powered by AOG (Act of Genius), an Emacs Org-mode static
  publishing system customized with the selfdotsend theme.  Posts are written
  in Org files and published through an editor-first workflow, so technical
  content can be iterated quickly without leaving Emacs.

  It is deployed on GitHub Pages and reflects a strong preference for
  open-source tooling, version-controlled content, and reproducible publishing.
