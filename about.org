* About Self dot send
:PROPERTIES:
:CUSTOM_ID: about-self-dot-send
:END:

Self dot send is a computing blog focused on programming languages, software
design, data structures, systems programming, and applied AI topics such as
large language models and speech recognition. [[https://selfdotsend.com][selfdotsend.com]]

It publishes hands-on tutorials, including editor internals like gap buffers,
and practical notes on Unix tooling, ARM assembly, and day-to-day engineering
work.

Most posts come from problems I ran into while building software, then wrote up
as notes others can reuse.

* About this Blog
:PROPERTIES:
:CUSTOM_ID: about-this-blog
:END:

This blog is powered by AOG (Act of Genius), a fork of
[[https://github.com/kelvinh/org-page][org-page]], an Emacs Org-mode static
publishing system, customized with the metacritical/selfdotsend theme.
[[https://github.com/metacritical/AOG][AOG on GitHub]]

Posts are written in Org files and published from an editor-first workflow, so
I can iterate quickly without leaving Emacs.
